

Hospitalito 🏥
A Robust Health Management System built with Java and Spring Boot.

📝 Description
Hospitalito is a backend application designed to streamline healthcare administration. It manages critical relationships between patients, doctors, and appointments, ensuring data integrity and a seamless user experience for clinic staff.

🚀 Key Features
Patient Management: Full CRUD operations for patient records.

Doctor Directory: Management of medical staff and specialties.

Appointment Scheduling: Business logic to prevent scheduling conflicts.

RESTful API: Clean and documented endpoints for frontend integration.

🛠️ Tech Stack
Language: Java 17.

Framework: Spring Boot 3.5.6.

Data Access: Spring Data JPA.

Database: MySQL.

Tooling: Lombok (for clean code), Maven (dependency management).

🧪 Testing & Quality
Following Test-Driven Development (TDD) principles, this project includes:

Unit Testing: Comprehensive tests using JUnit and Mockito to ensure business logic reliability.

Integration Testing: Validating the persistence layer with Spring Data JPA.

⚙️ Getting Started
Prerequisites
JDK 17 or higher.

Maven 3.x.

MySQL Server.
Installation & Running
Clone the repository:

Bash
git clone https://github.com/[TU_USUARIO]/hospitalito.git
Configure the database: Update src/main/resources/application.properties with your MySQL credentials.

Build the project:

Bash
./mvnw clean install
Run the application:

Bash
./mvnw spring-boot:run
